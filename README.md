## Objetivo
Aplicação de catálogo de produtos em uma estrutura de arquivos tripartida (**HTML, CSS e JS**).

- Deve permitir que o usuário gerencie uma lista de itens em tempo real.

---

## A interface deve conter:
- Um formulário para entrada de novos dados (**nome, categoria e preço**).
- Uma área de exibição que gere automaticamente *cards* ou linhas de tabela para cada item cadastrado.
- Uma seção de inteligência de dados que apresente métricas calculadas automaticamente:
  - Valor total do inventário  
  - Preço médio dos produtos  
  - Atualização instantânea a cada nova inserção ou remoção de item
